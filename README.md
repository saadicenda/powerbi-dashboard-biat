# 📊 Power BI Dashboard - BIAT

## 🎯 Objectif
Ce projet a pour objectif de développer un tableau de bord interactif en temps réel permettant de :

- 📈 Analyser la charge de travail des employés  
  (identifier les situations de sous-charge et de surcharge)

- 👥 Évaluer les besoins en effectifs de la banque  

- 📱 Proposer une version optimisée du tableau de bord pour les appareils mobiles  

Pour plus de détails sur la démarche, les indicateurs et les résultats, veuillez consulter le rapport du projet.

---

## 🛠️ Technologies utilisées

- Power BI Desktop / Power BI Service / Power Automate  
- Talend (ETL)  
- Oracle Database  

---

## 📂 Contenu du projet

- `PFE2025_VF.pbix` : fichier principal du tableau de bord  
- `Cenda_Rapport_VF.pdf` : rapport détaillé du projet  

---

## 📊 Fonctionnalités principales

- Suivi en temps réel des indicateurs de performance  
- Visualisation dynamique de la charge de travail  
- Aide à la prise de décision pour la gestion des ressources humaines  
- Version mobile du tableau de bord
